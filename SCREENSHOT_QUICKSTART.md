# Quick Start: Capture Screenshots

> **For Windows Users Only** - This is a WPF desktop application that requires Windows to run.

## Prerequisites
- Windows 10 or 11
- .NET 8.0 SDK installed

## Quick Steps

### 1. Build the Application
```bash
dotnet build
```

### 2. Run the Screenshot Guide
Choose one:

**PowerShell (Recommended):**
```powershell
.\Take-Screenshots.ps1
```

**Command Prompt:**
```cmd
.\Take-Screenshots.bat
```

### 3. Follow the Interactive Guide
The script will:
- Walk you through each of the 20 required screenshots
- Provide step-by-step instructions for each screen
- Tell you exactly what to capture and where to save it

### 4. Use Windows Snipping Tool
Press **Win + Shift + S** when prompted to capture each screenshot.

### 5. Save Screenshots
Save each file in the `screenshots/` folder with the name provided by the guide.

## Tips

✅ Use 1920x1080 or higher resolution  
✅ Capture full windows with title bars  
✅ Use demo accounts only - no real data  
✅ Save as PNG format  
✅ Disable desktop notifications before starting
